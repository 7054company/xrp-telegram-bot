# XRP Telegram bot

Built With :

- telegraf
- sequelize


## Minimum Requirement

- NodeJS 8.x >=
- npm 4.x >=
- Redis 3.x >=
- Mysql Server

## Install


```
$ npm i -g sequelize-cli
$ npm i
$ sequelize db:migrate
$ node index
```
